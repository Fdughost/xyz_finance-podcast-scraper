# 播客监控与日报系统 - 交付说明 (Excel 增强版)

## 项目概述

本系统是为播客《随基漫步》量身定制的小宇宙平台播客监控解决方案。系统能够自动化地抓取小宇宙平台上指定播客的核心数据，包括订阅数、最新单集标题、播放量、评论数、时长等，并生成专业的 Excel 格式日报，帮助您快速掌握竞品动态和行业趋势。

## 核心功能

本系统实现了以下核心功能模块：

### 1. 数据爬取模块（podcast_crawler.py）

该模块负责从小宇宙平台网页端抓取播客数据。系统采用了“网页深度扫描”技术，通过深度解析 HTML 页面并结合正则表达式，确保能稳定获取以下字段：
- **节目基础信息**：标题、简介、分类、主播/主理人、总订阅数。
- **最新单集信息**：单集标题（已清洗冗余简介）、上线日期、**播放量**、**评论数**、**时长**。

爬虫模块设计了智能的重试机制和请求延迟控制，确保在网络不稳定或目标服务器响应慢的情况下也能稳定运行。

### 2. 数据存储模块（data_manager.py）

数据存储模块使用 SQLite 轻量级数据库来持久化存储每日采集的数据。
- **持久化存储**：所有抓取的数据都会存入数据库，支持长期追踪。
- **历史对比**：系统能够自动对比今日与昨日的数据变化（如订阅数增减），并在报表中体现。

### 3. Excel 报表生成模块（excel_generator.py）

这是本次更新的核心模块，替代了原有的 Word 报表：
- **结构化输出**：首行固定为字段名称，每行对应一个播客的完整数据。
- **专业格式**：自动调整列宽，支持长文本（如简介、标题）的自动换行，方便在移动端或电脑端直接阅读。
- **数据分析友好**：Excel 格式可直接用于制作透视图、筛选和排序。

### 4. 主程序（main.py）

主程序串联了上述模块，实现了一键式的自动化流程。运行后，系统会依次完成数据爬取、数据库存储和 Excel 报表生成。

## 使用建议

1. **运行频率**：建议每天早上 8-10 点运行一次，以获取最新的单集播放量和更新状态。
2. **扩展名单**：如果您发现新的竞品，只需在 `config.json` 中添加其小宇宙 ID 即可。
3. **数据应用**：生成的 Excel 报表可以直接作为代运营周报/日报的附件，或者用于内部选题分析。

## 注意事项

- **播放量获取**：由于小宇宙网页版有时会对播放量进行动态加载或隐藏，系统采用了多重备选解析方案。如果某些单集刚发布不久，播放量可能显示为 0 或较低。
- **依赖库**：请确保安装了 `pandas` 和 `openpyxl` 以支持 Excel 生成。
